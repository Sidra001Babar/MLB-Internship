# OpenCV Basics

## What is OpenCV?

**OpenCV (Open Source Computer Vision Library)** is an open-source library used for **Computer Vision**, **Image Processing**, and **Machine Learning** tasks. It provides hundreds of optimized algorithms for working with images and videos.

### Common Applications
- Image processing
- Object detection
- Face detection and recognition
- Image classification
- Motion tracking
- Video analysis
- Medical image processing
- Self-driving cars

### Installing OpenCV

```bash
pip install opencv-python
```

### Importing OpenCV

```python
import cv2
```

# Reading and Displaying Images

OpenCV reads an image using the `cv2.imread()` function.

### Syntax

```python
image = cv2.imread("image.jpg")
```

### Example

```python
import cv2

image = cv2.imread("cat.jpg")

cv2.imshow("Cat Image", image)

cv2.waitKey(0)

cv2.destroyAllWindows()
```

### Explanation

- `cv2.imread()` reads the image from disk.
- `cv2.imshow()` displays the image in a window.
- `cv2.waitKey(0)` waits until a key is pressed.
- `cv2.destroyAllWindows()` closes all OpenCV windows.

---

# Saving Images

After editing or processing an image, it can be saved using `cv2.imwrite()`.

### Syntax

```python
cv2.imwrite("output.jpg", image)
```

### Example

```python
import cv2

image = cv2.imread("cat.jpg")

cv2.imwrite("saved_cat.jpg", image)
```

### Explanation

- `cv2.imwrite()` saves the image to the specified file.
- If the file already exists, it will be overwritten.

# Image Properties (Height, Width, Channels)

The `shape` attribute returns the dimensions of an image.

### Syntax

```python
image.shape
```

### Example

```python
import cv2

image = cv2.imread("cat.jpg")

print(image.shape)
```

### Output

```python
(720, 1280, 3)
```

### Meaning

- **720** → Height (number of rows)
- **1280** → Width (number of columns)
- **3** → Number of color channels (Blue, Green, Red)

You can also access them separately.

```python
height = image.shape[0]
width = image.shape[1]
channels = image.shape[2]

print("Height:", height)
print("Width:", width)
print("Channels:", channels)
```

# BGR vs RGB

Although most image-processing libraries use the **RGB** color format, OpenCV uses **BGR** by default.

## RGB Order

```
Red → Green → Blue
```

## OpenCV (BGR) Order

```
Blue → Green → Red
```

### Why is this important?

If an image is displayed using another library like Matplotlib without converting it, the colors will appear incorrect.

### Convert BGR to RGB

```python
import cv2

image = cv2.imread("cat.jpg")

rgb_image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
```

### Explanation

- `cv2.cvtColor()` changes the color format.
- `cv2.COLOR_BGR2RGB` converts an OpenCV image into RGB format.

# Grayscale Images

A grayscale image contains only **intensity (brightness)** values instead of color information.

Each pixel has **one value** ranging from:

- **0** → Black
- **255** → White

### Read an Image in Grayscale

```python
import cv2

gray = cv2.imread("cat.jpg", cv2.IMREAD_GRAYSCALE)

cv2.imshow("Gray Image", gray)

cv2.waitKey(0)

cv2.destroyAllWindows()
```

### Convert a Color Image to Grayscale

```python
import cv2

image = cv2.imread("cat.jpg")

gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
```

### Explanation

- `cv2.IMREAD_GRAYSCALE` loads the image directly as grayscale.
- `cv2.COLOR_BGR2GRAY` converts a color image into grayscale after loading.
