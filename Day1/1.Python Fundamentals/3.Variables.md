## Global Variables
https://www.w3schools.com/python/python_variables_global.asp

### Def
    Variable outside the function
### Def of Local Variable
    Variable inside the function
### Note:
    if there is same name of global and local variable, nothing will override, local will remain local inside that funcation and global will remain golbal outside the function, but inside the function that global variable will not work if same name of local variable is present inside the function

### Global Keyword:
   1.  Use global keyword to create make a local variable global inside a function

   2. Use global keyword to overrride the global variables' value inside a function 

```python
x = "awesome"

def myfunc():
    global x
    x = "fantastic"

myfunc()

print("Python is " + x)
```
