| Feature | List | Tuple | Dictionary | Set |
|---------|------|-------|------------|-----|
| **Definition** | Ordered collection of items | Ordered, immutable collection of items | Collection of key-value pairs | Unordered collection of unique items |
| **Syntax** | `[]` | `()` | `{key: value}` | `{}` or `set()` |
| **Ordered** | Yes | Yes | Yes (Python 3.7+) | No |
| **Mutable** | Yes | No | Yes | Yes |
| **Allows Duplicates** | Yes | Yes | Keys: No, Values: Yes | No |
| **Indexed** | Yes | Yes | No (accessed by keys) | No |
| **Key-Value Pairs** | No | No | Yes | No |
| **Unique Elements** | No | No | Keys are unique | Yes |
| **Supports Mixed Data Types** | Yes | Yes | Yes | Yes |
| **Supports Slicing** | Yes | Yes | No | No |
| **Common Methods** | `append()`, `remove()`, `sort()` | `count()`, `index()` | `keys()`, `values()`, `items()`, `get()` | `add()`, `remove()`, `union()`, `intersection()` |
| **Best Use Case** | Storing data that changes frequently | Storing fixed/read-only data | Storing and retrieving data using keys | Storing unique items and performing set operations |
| **Example** | `[10, 20, 30]` | `(10, 20, 30)` | `{"name": "Ali", "age": 20}` | `{10, 20, 30}` |
