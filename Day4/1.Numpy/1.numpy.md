# NumPy

## What is NumPy?
NumPy (Numerical Python) is a powerful Python library used for numerical computing. It is mainly used to create and work with **arrays**, which are faster and more efficient than Python lists.

Apart from arrays, NumPy provides functions for:
- Matrix Operations
- Mathematical and Statistical Calculations
- Linear Algebra: Linear Algebra deals with vectors, matrices, and systems of equations. NumPy provides many functions for performing these operations efficiently.
- Fourier Transform: 
#### What is Fourier Transform?

A **Fourier Transform** is a mathematical technique that breaks a complex signal into its individual frequencies.

In simple words:

> It tells us **what frequencies (or notes) are present in a signal.**

Instead of asking:

> "How does the signal change over time?"

it asks:

> "Which frequencies make up this signal?"

---

Imagine someone plays a piano and presses three keys at the same time.

🎹

- Note C
- Note E
- Note G

A Fourier Transform separates that sound and tells you:

```
Frequency 1 → Note C
Frequency 2 → Note E
Frequency 3 → Note G
```

So it converts:

```
One mixed sound
        ↓
Separate frequencies
```

# Why is NumPy Faster Than Python Lists?

The biggest reason is **memory storage**.

### Python List

A Python list stores references (addresses) to objects.

```
Memory

1000 → 5
2050 → 8
5010 → 12
7020 → 20
```

Since the elements are stored at different memory locations, the CPU needs more time to access them.

---

### NumPy Array

NumPy stores all elements in **continuous memory locations**.

```
Memory

1000 → 5
1004 → 8
1008 → 12
1012 → 20
```

Because the data is stored continuously:

- CPU cache works efficiently.
- Access is much faster.
- Mathematical operations become highly optimized.

This concept is known as **Locality of Reference**.

---

# What is Locality of Reference?

**Locality of Reference** means storing related data close together in memory.

Since NumPy arrays store elements consecutively, the processor can quickly access nearby elements without repeatedly searching different memory locations.

Example:

```
Python List

1000
5000
9000
15000
```

```
NumPy Array

1000
1004
1008
1012
```

Continuous storage greatly improves performance.

# Checking the Number of Dimensions

Use the **`ndim`** attribute to find the number of dimensions of an array.

```python
import numpy as np

a = np.array(42)
b = np.array([1, 2, 3])
c = np.array([[1, 2], [3, 4]])
d = np.array([[[1, 2], [3, 4]]])

print(a.ndim)
print(b.ndim)
print(c.ndim)
print(d.ndim)
```

**Output**

```
0
1
2
3
```

**Explanation**

- `0` → Scalar
- `1` → Vector (1-D)
- `2` → Matrix (2-D)
- `3` → Collection of matrices (3-D)

---

# Higher Dimensional Arrays

NumPy allows you to create arrays with any number of dimensions using the `ndmin` parameter.

```python
import numpy as np

arr = np.array([1, 2, 3, 4], ndmin=5)

print(arr)
print(arr.ndim)
```

**Output**

```
[[[[[1 2 3 4]]]]]
5
```

### Explanation

- `ndmin=5` forces the array to have **5 dimensions**.
- NumPy adds extra dimensions of size **1** until the required dimensions are reached.

Structure:

```
Dimension 1
 └── Dimension 2
      └── Dimension 3
           └── Dimension 4
                └── [1 2 3 4]  ← Dimension 5
```

# NumPy Array Indexing

## What is Array Indexing?

**Array indexing** means accessing a specific element in a NumPy array using its **index (position)**.

> **Note:** NumPy uses **zero-based indexing**, which means:
>
> - First element → Index `0`
> - Second element → Index `1`
> - Third element → Index `2`

---

# Accessing Elements in a 1-D Array

## Example 1: Access the First Element

```python
import numpy as np

arr = np.array([1, 2, 3, 4])

print(arr[0])
```

### Explanation

- `arr[0]` accesses the element at index `0`.
- The first element is `1`.

---

## Example 2: Access the Second Element

```python
import numpy as np

arr = np.array([1, 2, 3, 4])

print(arr[1])
```

**Output**

```
2
```

### Explanation

- `arr[1]` accesses the element at index `1`.
- The second element is `2`.

---

## Example 3: Add the Third and Fourth Elements

```python
import numpy as np

arr = np.array([1, 2, 3, 4])

print(arr[2] + arr[3])
```

**Output**

```
7
```

### Explanation

- `arr[2]` = `3`
- `arr[3]` = `4`
- `3 + 4 = 7`

# Using the Step Value

The **step** specifies how many positions to move after selecting an element.

**Syntax**

```python
array[start:end:step]
```

---

## Example 1: Return Every Other Element

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6, 7])

print(arr[1:5:2])
```

**Output**

```
[2 4]
```

### Explanation

- Start at index `1`.
- End before index `5`.
- Skip one element each time (`step = 2`).

```
Index : 0 1 2 3 4 5 6
Value : 1 2 3 4 5 6 7
          ✓   ✓
```

---

## Example 2: Return Every Other Element from the Entire Array

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5, 6, 7])

print(arr[::2])
```

**Output**

```
[1 3 5 7]
```

### Explanation

- Start from the beginning.
- Go to the end.
- Select every second element.

# NumPy Array Reshaping

## What is Reshaping?

**Reshaping** means changing the **shape (dimensions)** of an array **without changing its data**.

The **shape** of an array tells us how many elements are present in each dimension.

Reshaping allows you to:

- Convert a 1-D array into a 2-D or 3-D array.
- Increase or decrease the number of dimensions.
- Change the arrangement of elements while keeping the total number of elements the same.

---

# Reshaping from 1-D to 2-D

A **1-D array** can be converted into a **2-D array** by specifying the number of rows and columns.

For example:

- A 1-D array containing **12 elements** can be reshaped into:
  - **4 rows × 3 columns**
  - **2 rows × 6 columns**
  - **3 rows × 4 columns**

As long as the total number of elements remains **12**, the reshape operation is valid.

---

# Reshaping from 1-D to 3-D

A **1-D array** can also be converted into a **3-D array**.

A 3-D array is represented as:

```
(Number of Arrays, Number of Rows, Number of Columns)
```

For example:

```
(2, 3, 2)
```

means:

- 2 arrays
- Each array contains 3 rows
- Each row contains 2 columns

Total elements required:

```
2 × 3 × 2 = 12
```

Since the original array also contains 12 elements, reshaping is possible.

---

# Can We Reshape into Any Shape?

**Yes**, but only if the total number of elements remains the same.

The following formula must always be true:

```
Rows × Columns × ... = Total Number of Elements
```

### Valid Example

Original array:

```
8 elements
```

New shape:

```
2 × 4 = 8
```

✔ Valid because both contain **8 elements**.

---

### Invalid Example

Original array:

```
8 elements
```

New shape:

```
3 × 3 = 9
```

✘ Invalid because **9 elements** are required, but only **8 elements** are available.

NumPy will raise a **ValueError** in this case.

---

# Does `reshape()` Return a Copy or a View?

By default, `reshape()` returns a **view** of the original array whenever possible.

### View

- Shares the same memory as the original array.
- Changing one may affect the other.

### Copy

- Creates a completely separate array.
- Changes do not affect the original array.

Most reshape operations return a **view**, making them memory efficient.

# Unknown Dimension (`-1`)

NumPy allows **one dimension** to be specified as **`-1`**.

This means:

> "Automatically calculate the size of this dimension."

NumPy computes the missing dimension based on the total number of elements.

For example:

```
Original array

8 elements
```

If two dimensions are specified as:

```
2 × 2 × ?
```

NumPy automatically calculates:

```
2 × 2 × 2 = 8
```

So the missing dimension becomes **2**.

### Important Rule

Only **one** dimension can be `-1`.

Using `-1` for multiple dimensions is not allowed because NumPy would not know how to calculate both values.

---

# Flattening an Array

**Flattening** means converting a **multidimensional array** into a **1-D array**.

For example:

Before flattening:

```
[
 [1 2 3]
 [4 5 6]
]
```

After flattening:

```
[1 2 3 4 5 6]
```

Flattening is useful when:

- Preparing data for Machine Learning.
- Converting matrices into vectors.
- Simplifying data for further processing.

A common way to flatten an array is by using:

```
reshape(-1)
```

Here, `-1` tells NumPy to create a **one-dimensional array** containing all the elements.
