## Data Cleaning Steps

The following data cleaning steps were performed on the dataset:

- Checked for missing values and handled them appropriately.
- Removed duplicate records to ensure data quality.
- Renamed one or more columns for better readability.
- Created a new column named **Average_Score** by calculating the average marks across all subjects.
- Created a new column named **Performance** based on the students' average scores to categorize their overall performance.

## Visualizations Created

The following visualizations were created to analyze the dataset:

- **Bar Chart:** Displayed the students based on their average scores for easy comparison.
- **Histogram:** Showed the distribution of average scores to understand how student performance is spread.
- **Scatter Plot:** Compared Python and Machine Learning marks to identify any relationship between the two subjects.
- **Pie Chart:** Displayed the distribution of students across different performance categories.
- **Box Plot:** Visualized the spread of marks across all subjects and helped identify any outliers.
