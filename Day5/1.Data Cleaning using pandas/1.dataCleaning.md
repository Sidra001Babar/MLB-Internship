## 1. Missing Value Handling Techniques in Pandas
https://www.youtube.com/@campusx-official/playlists

Note: all these techniques are from the play list of 100 days of machine learning but AI tools suggest some more techniques like Forward Fill (FFill),Backward Fill (BFill),Linear Interpolation,Polynomial Interpolation,Time-Based Interpolation,Regression Imputationand Domain-Specific Imputation etc

| Technique | Description | Best Used When | Advantages | Disadvantages |
|-----------|-------------|----------------|------------|---------------|
| **Detect Missing Values** | Identify missing values in the dataset before handling them. | Initial data exploration (EDA). | Helps understand data quality. | Does not solve the problem. |
| **Remove Rows** | Delete rows containing missing values. | Only a few rows have missing data. | Simple and fast. | Loss of data; may introduce bias. |
| **Remove Columns** | Delete columns containing missing values. | A column has too many missing values and is not important. | Reduces noise in data. | Important information may be lost. |
| **Threshold-Based Removal** | Remove rows or columns only if missing values exceed a specified percentage. | Some missing values are acceptable. | Preserves more data than complete removal. | Choosing the right threshold can be difficult. |
| **Fill with Constant Value** | Replace missing values with a fixed value such as `0`, `-1`, `"Unknown"`, or `"N/A"`. | Categorical data or when missing values have a specific meaning. | Easy to implement; no data loss. | May introduce unrealistic values. |
| **Fill with Mean** | Replace missing numerical values with the average of the column. | Numerical data with a normal distribution and few outliers. | Maintains the column's average. | Sensitive to outliers. |
| **Fill with Median** | Replace missing values with the middle value of the column. | Numerical data with skewed distribution or outliers. | Robust to outliers. | Does not consider relationships between features. |
| **Fill with Mode** | Replace missing values with the most frequently occurring value. | Categorical data. | Maintains existing categories. | Can overrepresent the most common value. |
| **Random Value Imputation** | Replace missing values with randomly selected existing values from the same column. | When preserving the original distribution is important. | Maintains variance. Distribution of data b same rhy gi| Results vary between executions. |
| **K-Nearest Neighbors (KNN) Imputation** | Estimate missing values using similar observations. | Features have strong relationships. | Often produces accurate estimates. | Computationally expensive; requires tuning. |
| **Iterative Imputation** | Skip for now

# Recommended Solutions

| Data Type / Situation | Recommended Method |
|------------------------|--------------------|
| Few missing rows | Remove Rows |
| Column with >70% missing values | Remove Column |
| Numerical data (Normal Distribution) | Mean |
| Numerical data (Skewed/Outliers) | Median |
| Categorical data | Mode or Constant Value |
| Correlated Features | KNN or Regression Imputation |
| Large & Complex Dataset | Machine Learning Imputation |
| Statistical Research | Multiple Imputation |
| Missing Values Have Meaning | Leave As-Is or Fill with "Unknown" |

# 2. Removing Duplicate Records

Duplicate records can be removed using the `drop_duplicates()` method.

### Remove Duplicate Rows

```python
df = df.drop_duplicates()
```

### Remove Duplicates Based on Specific Columns

```python
df = df.drop_duplicates(subset=["Name"])
```

### Keep the Last Occurrence

```python
df = df.drop_duplicates(
    subset=["Name"],
    keep="last"
)
```

### Remove All Duplicates

```python
df = df.drop_duplicates(
    subset=["Name"],
    keep=False
)
```

# 5. Creating New Columns

New columns can be created using existing data or custom logic.

### Direct Assignment

```python
df["Country"] = "Pakistan"
```

### Copy Existing Column

```python
df["Age_Copy"] = df["Age"]
```

### Mathematical Operation

```python
df["Total"] = df["Math"] + df["Science"]
```

### Conditional Column

```python
import numpy as np

df["Result"] = np.where(
    df["Marks"] >= 50,
    "Pass",
    "Fail"
)
```

### Using a Custom Function

```python
def grade(marks):
    if marks >= 90:
        return "A"
    else:
        return "B"

df["Grade"] = df["Marks"].apply(grade)
```

### Concatenate Columns

```python
df["Full_Name"] = (
    df["First_Name"] +
    " " +
    df["Last_Name"]
)
```

# 7. Filtering Data
Trick: logics as of database queries

Filtering is used to select rows based on conditions.

### Filter Rows with a Single Condition

```python
filtered_df = df[
    df["Marks"] >= 50
]
```

### Filter Using Multiple Conditions

```python
filtered_df = df[
    (df["Marks"] >= 50) &
    (df["Age"] > 18)
]
```

### Filter Using OR Condition

```python
filtered_df = df[
    (df["Marks"] >= 90) |
    (df["Age"] < 18)
]
```

### Filter Using `isin()`

```python
filtered_df = df[
    df["City"].isin(
        ["Lahore", "Karachi"]
    )
]
```

### Filter Missing Values

```python
filtered_df = df[
    df["Marks"].isnull()
]
```

### Filter Non-Missing Values

```python
filtered_df = df[
    df["Marks"].notnull()
]
```
