# Features and Target Variable

## What are Features?

Features are the **input variables** (independent variables) used by a machine learning model to make predictions. They contain the information that helps the model learn patterns from the data.

Features are commonly represented by the variable **`X`**.

### Example

Suppose we have the following dataset for predicting house prices:

| House Size (sq ft) | Bedrooms | Age (Years) | Price |
|-------------------:|---------:|------------:|------:|
| 1200 | 2 | 10 | 150000 |
| 1800 | 3 | 5 | 250000 |
| 2500 | 4 | 2 | 400000 |

The features are:

- House Size
- Bedrooms
- Age

These columns are used as inputs to predict the house price.

---

## What is the Target Variable?

The **target variable** (also called the **label**, **output variable**, or **dependent variable**) is the value that the machine learning model is trying to predict.

The target variable is commonly represented by the variable **`y`**.

### Example

Using the same dataset:

| House Size | Bedrooms | Age | **Price (Target)** |
|------------|----------|-----|--------------------|
| 1200 | 2 | 10 | 150000 |
| 1800 | 3 | 5 | 250000 |
| 2500 | 4 | 2 | 400000 |

Here, **Price** is the target variable because it is the value we want the model to predict.

---

# Features and Target in Text Classification

Consider the following dataset:

| Text | Label |
|------|-------|
| I love this movie. | Positive |
| This product is terrible. | Negative |
| Amazing experience! | Positive |

### Features (`X`)

The **Text** column contains the input data. After preprocessing and feature extraction (e.g., TF-IDF), it becomes the feature matrix **`X`**.

### Target Variable (`y`)

The **Label** column contains the output classes that the model learns to predict.

Possible labels:

- Positive
- Negative

---

# Difference Between Features and Target Variable

| Features (X) | Target Variable (y) |
|--------------|---------------------|
| Input variables used for prediction. | Output variable to be predicted. |
| Also called independent variables. | Also called dependent variable or label. |
| There can be one or many feature columns. | Usually a single column. |
| Used by the model to learn patterns. | Used to evaluate whether predictions are correct. |
