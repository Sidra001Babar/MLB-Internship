# Python JSON

## What is JSON?

**JSON (JavaScript Object Notation)** is a lightweight format used for **storing** and **exchanging** data.

Although JSON is based on JavaScript object syntax, it is a **language-independent** format that is supported by many programming languages, including Python.

JSON is commonly used for:

- APIs (sending and receiving data)
- Configuration files
- Data storage
- Communication between applications

Example of JSON data:

```json
{
    "name": "John",
    "age": 30,
    "city": "New York"
}
```

---

# JSON in Python

Python provides a built-in module called **`json`** for working with JSON data.

Before using it, import the module.

```python
import json
```

The `json` module allows you to:

- Convert JSON → Python
- Convert Python → JSON
- Format JSON for better readability

---

# Parse JSON (Convert JSON to Python)

If you have JSON data stored as a **string**, you can convert it into a Python object using:

```python
json.loads()
```

The word **loads** means:

> **Load JSON from a string.**

---

## Syntax

```python
json.loads(json_string)
```

---

## Example

```python
import json

# JSON string
x = '{ "name":"John", "age":30, "city":"New York"}'

# Convert JSON to Python dictionary
y = json.loads(x)

print(y["age"])
```

### Output

```python
30
```

---

## How It Works

JSON string:

```json
{
    "name":"John",
    "age":30,
    "city":"New York"
}
```

After:

```python
y = json.loads(x)
```

Python converts it into a **dictionary**.

```python
{
    "name": "John",
    "age": 30,
    "city": "New York"
}
```

Now you can access values like a normal dictionary.

```python
print(y["name"])
```

Output

```python
John
```

---

# Convert Python to JSON

If you have a Python object, convert it into a JSON string using:

```python
json.dumps()
```

The word **dumps** means:

> **Dump Python data into a JSON string.**

---

## Syntax

```python
json.dumps(python_object)
```

---

## Example

```python
import json

x = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

y = json.dumps(x)

print(y)
```

### Output

```json
{"name": "John", "age": 30, "city": "New York"}
```

The output is a **JSON string**.

---

# `loads()` vs `dumps()`

| Method | Converts From | Converts To |
|---------|---------------|-------------|
| `json.loads()` | JSON string | Python object |
| `json.dumps()` | Python object | JSON string |

Remember:

- **loads** → JSON → Python
- **dumps** → Python → JSON

# Python Objects That Can Be Converted to JSON

Python can convert the following object types into JSON.

| Python Type | JSON Type |
|-------------|-----------|
| `dict` | Object |
| `list` | Array |
| `tuple` | Array |
| `str` | String |
| `int` | Number |
| `float` | Number |
| `True` | `true` |
| `False` | `false` |
| `None` | `null` |

---

# Examples

## Dictionary

```python
import json

print(json.dumps({"name": "John", "age": 30}))
```

Output


## List


Output


## Tuple

```python

```

Output

```json
["apple", "banana"]
```

> **Note:** JSON has no tuple type, so tuples become arrays.

---

## String

```python

```

Output

```json
"hello"
```

---

## Integer

```python

```

Output

```json
42
```

---

## Float

```python
print(json.dumps(31.76))
```

Output

```json
31.76
```

---

## Boolean (`True`)

```python
print(json.dumps(True))
```

Output

```json
true
```

Notice that JSON uses lowercase:

```text
true
```

instead of

```text
True
```

---

## Boolean (`False`)

```python

```

Output

```json
false
```

---

## None


Output

```json
null
```

# Complex Python Object

JSON can also represent nested objects and arrays.

Example

```python
import json

x = {
    "name": "John",
    "age": 30,
    "married": True,
    "divorced": False,
    "children": ("Ann", "Billy"),
    "pets": None,
    "cars": [
        {
            "model": "BMW 230",
            "mpg": 27.5
        },
        {
            "model": "Ford Edge",
            "mpg": 24.1
        }
    ]
}

print(json.dumps(x))
```

### Output (single line)

```json
{"name":"John","age":30,"married":true,"divorced":false,"children":["Ann","Billy"],"pets":null,"cars":[{"model":"BMW 230","mpg":27.5},{"model":"Ford Edge","mpg":24.1}]}
```

Notice:

- Tuple becomes an array.
- `True` becomes `true`.
- `False` becomes `false`.
- `None` becomes `null`.

# Formatting JSON

By default, `json.dumps()` produces JSON on a single line.

For better readability, use the **`indent`** parameter.

---

## Using `indent`

Example

```python
import json

print(json.dumps(x, indent=4))
```

### Output

```json
{
    "name": "John",
    "age": 30,
    "married": true,
    "divorced": false,
    "children": [
        "Ann",
        "Billy"
    ],
    "pets": null
}
```

The `indent=4` argument tells Python to use **4 spaces** for each indentation level.

---

# Using `separators`

The **`separators`** parameter changes how keys and values are separated.

Default separators are:

```python
(", ", ": ")
```

Meaning:

- Comma followed by a space separates items.
- Colon followed by a space separates keys and values.

Example output:

```json
{
    "name": "John",
    "age": 30
}
```

---

## Custom Separators

Example

```python
import json

print(json.dumps(x, indent=4, separators=(". ", " = ")))
```

Output

```text
{
    "name" = "John".
    "age" = 30
}
```

Here:

- `". "` replaces the comma separator.
- `" = "` replaces the colon separator.

This formatting is mainly used for custom display purposes and is **not standard JSON**.

# Key Points

- JSON stands for **JavaScript Object Notation**.
- JSON is a text-based format used for storing and exchanging data.
- Import the `json` module before working with JSON.
- Use **`json.loads()`** to convert a JSON string into a Python object (usually a dictionary).
- Use **`json.dumps()`** to convert a Python object into a JSON string.
- Python types such as dictionaries, lists, tuples, strings, numbers, booleans, and `None` can all be converted to JSON.
- Use the `indent` parameter to produce formatted, human-readable JSON.
- Use the `separators` parameter to customize how keys and values are separated.
