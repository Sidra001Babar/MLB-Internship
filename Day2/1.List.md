# Python Lists

## What is a List?

A **list** is a built-in Python data type used to store **multiple items in a single variable**.

Lists are one of Python's **four built-in collection data types**:

- **List** → Ordered, changeable, allows duplicates.
- **Tuple** → Ordered, unchangeable, allows duplicates.
- **Set** → Unordered, changeable,unchangeable*(frozenset), no duplicates.
- **Dictionary** → Ordered**, changeable, stores key-value pairs.

---

# Creating a List

Lists are created using **square brackets `[]`**.

### Syntax

```python
mylist = ["apple", "banana", "cherry"]
print(mylist)
```

### Output

```python
['apple', 'banana', 'cherry']
```

You can also create a list using the `list()` constructor.

```python
thislist = list(("apple", "banana", "cherry"))
print(thislist)
```

### Output

```python
['apple', 'banana', 'cherry']
```

> **Note:** When using `list()`, notice the **double parentheses**:
>
> - Outer parentheses belong to `list()`
> - Inner parentheses create a tuple that is converted into a list.

---

# Properties of Python Lists

Python lists have three important characteristics:

1. Ordered
2. Changeable (Mutable)
3. Allow Duplicates

---

# 1. Ordered

Lists maintain the order in which items are added.

```python
fruits = ["apple", "banana", "cherry"]

print(fruits)
```

Output

```python
['apple', 'banana', 'cherry']
```

If you add another item:

```python
fruits.append("orange")

print(fruits)
```

Output

```python
['apple', 'banana', 'cherry', 'orange']
```

The new item is always added **at the end**.

> **Note:** Some list methods (like `sort()` and `reverse()`) can change the order.

---

# 2. Changeable (Mutable)

Unlike tuples, lists can be modified after creation.

You can:

- Change items
- Add items
- Remove items

Example

```python
fruits = ["apple", "banana", "cherry"]

fruits[1] = "orange"

print(fruits)
```

Output

```python
['apple', 'orange', 'cherry']
```

---

# 3. Allow Duplicates

Lists allow multiple items with the same value.

Example

```python
fruits = ["apple", "banana", "cherry", "apple", "cherry"]

print(fruits)
```

Output

```python
['apple', 'banana', 'cherry', 'apple', 'cherry']
```

Duplicates are perfectly valid in Python lists.

# List Indexing

Every item in a list has an index.

Indexes start from **0**.

| Index | Value |
|-------:|-------|
| 0 | apple |
| 1 | banana |
| 2 | cherry |

Example

```python
fruits = ["apple", "banana", "cherry"]

print(fruits[0])
print(fruits[1])
print(fruits[2])
```

Output

```python
apple
banana
cherry
```

# Python - Access List Items - By Positive Indexing, Negative indexing, Slicing

## Accessing List Items

Lists are **indexed**, which means every item has a position called an **index**.

The first item always starts at **index `0`**.

### Example

```python
thislist = ["apple", "banana", "cherry"]

print(thislist[1])
```

### Output

```python
banana
```

### Index Positions

| Index | Value |
|------:|-------|
| 0 | apple |
| 1 | banana |
| 2 | cherry |

> **Note:** Python indexing starts from **0**, not 1.

---

# Positive Indexing

Positive indexes count from **left to right**.

```text
Index:    0        1         2
        ----------------------------
List :  apple   banana   cherry
```

Example

```python
fruits = ["apple", "banana", "cherry"]

print(fruits[0])
print(fruits[1])
print(fruits[2])
```

### Output

```python
apple
banana
cherry
```

---

# Negative Indexing

Negative indexing starts counting from the **end of the list**.

| Negative Index | Value |
|--------------:|-------|
| -1 | cherry |
| -2 | banana |
| -3 | apple |

### Visualization

```text
Positive Index:   0        1         2
                 ------------------------
List:           apple   banana   cherry
                 ------------------------
Negative Index: -3       -2       -1
```

### Example

Print the last item.

```python
thislist = ["apple", "banana", "cherry"]

print(thislist[-1])
```

### Output

```python
cherry
```

Another example:

```python
print(thislist[-2])
```

Output

```python
banana
```

---

# Range of Indexes (List Slicing)

Instead of accessing one item, you can access **multiple items** using **slicing**.

### Syntax

```python
list[start: end] 
```
means [start index:end index - 1] [start: end] : in this case sliicing start from start index and ends at end index - 1
- **start** → Included
- **end** → Not included


This means Python returns items from **start** up to **end - 1**. Means it will skip last index 

---

## Example

```python
thislist = [
    "apple",
    "banana",
    "cherry",
    "orange",
    "kiwi",
    "melon",
    "mango"
]

print(thislist[2:5])
```

### Output

```python
['cherry', 'orange', 'kiwi']
```

### Explanation

```text
Index :   0       1       2        3       4       5       6
         -------------------------------------------------------
List :  apple  banana  cherry  orange   kiwi   melon   mango
                     ↑--------------------↑
                    start=2             end=5
```

Python includes:

- Index **2**
- Index **3**
- Index **4**

It **does not include index 5**.

---

# Omitting the Start Index

If the **start index is omitted**, Python starts from the beginning of the list.

### Syntax

```python
list[:end]
```

### Example

```python
thislist = [
    "apple",
    "banana",
    "cherry",
    "orange",
    "kiwi",
    "melon",
    "mango"
]

print(thislist[:4])
```

### Output

```python
['apple', 'banana', 'cherry', 'orange']
```

### Explanation

Python starts from index **0** and stops **before index 4**.

```text
0      1       2       3      4
------------------------------------
apple banana cherry orange kiwi
↑-------------------------↑
          Returned
```

---

# Omitting the End Index

If the **end index is omitted**, Python returns items until the end of the list.

### Syntax

```python
list[start:]
```

### Example

```python
thislist = [
    "apple",
    "banana",
    "cherry",
    "orange",
    "kiwi",
    "melon",
    "mango"
]

print(thislist[2:])
```

### Output

```python
['cherry', 'orange', 'kiwi', 'melon', 'mango']
```

### Explanation

Python starts at index **2** and continues until the last item.

---
Note: myList[:] it will return the complete list
# Range of Negative Indexes

You can also slice using **negative indexes**.

### Example

```python
thislist = [
    "apple",
    "banana",
    "cherry",
    "orange",
    "kiwi",
    "melon",
    "mango"
]

print(thislist[-4:-1])
```

### Output

```python
['orange', 'kiwi', 'melon']
```

### Explanation

Negative indexes:

| Index | Value |
|------:|-------|
| -7 | apple |
| -6 | banana |
| -5 | cherry |
| -4 | orange |
| -3 | kiwi |
| -2 | melon |
| -1 | mango |

Python starts at **-4** (orange) and stops **before -1** (mango).

```text
apple banana cherry orange kiwi melon mango
                   ↑-------------↑
                  -4           -1(not included)
```

Returned items:

- orange
- kiwi
- melon

# Check if an Item Exists

Use the **`in`** keyword to check whether a value exists in a list.

### Syntax

```python
item in list
```

It returns:

- `True` if the item exists.
- `False` if it does not.

---

## Example

```python
thislist = ["apple", "banana", "cherry"]

if "apple" in thislist:
    print("Yes, 'apple' is in the fruits list")
```

### Output

```python
Yes, 'apple' is in the fruits list
```

---

## Example When the Item Does Not Exist

```python
thislist = ["apple", "banana", "cherry"]

if "orange" in thislist:
    print("Found")
else:
    print("Not Found")
```

### Output

```python
Not Found
```

---

# Summary

| Operation | Syntax | Example |
|-----------|--------|---------|
| Access first item | `list[0]` | `fruits[0]` |
| Access second item | `list[1]` | `fruits[1]` |
| Access last item | `list[-1]` | `fruits[-1]` |
| Slice between indexes | `list[start:end]` | `fruits[2:5]` |
| From beginning | `list[:end]` | `fruits[:4]` |
| To the end | `list[start:]` | `fruits[2:]` |
| Negative slice | `list[-4:-1]` | `fruits[-4:-1]` |
| Check item exists | `item in list` | `"apple" in fruits` |

---

# Key Points

- List indexing starts at **0**.
- Negative indexing starts from the **end** of the list.
- Slicing returns a **new list**.
- The **start index is included**, but the **end index is excluded**.
- Omitting the start index means **start from the beginning**.
- Omitting the end index means **continue to the end**.
- Use the **`in`** keyword to check whether an item exists in a list.
